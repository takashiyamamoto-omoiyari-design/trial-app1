# 安全上のご注意〈必ずお守りください〉

人への危害や財産の損壊を防止するため、必ずお守りいただくことを説明しています。

![画像: 洗濯機に閉じ込められる、電源プラグからの発火]

誤った使い方をしたときに生じる危害や損害の程度を
区分して説明しています。

## 子供をドラムに入らせない
（ドラム内に閉じ込められて窒息したり、やけど・感電・けがをおう恐れがあります）

![画像: 子供がドラムの中に入ろうとしている様子]

- ✗ 子供だけで使わせない。

## 洗濯槽で絶対に乾燥しない
（油の酸化による発火のおそれ）

![画像: 洗濯槽]

- 油が付いたもの
- ふんわり加工
- ✗ ミシン油（ニスフィルム）
- ✗ 布の糸生地
- ✗ ニット・カンゾウ
- ✗ シンナー・ガソリン
- ✗ ポリエステル織物
- ✗ スポンジ材
- ✗ ビニール袋
- ✗ セロハン・不織布

## 電源プラグやコードを正しく使う
① 定格15 A以上のコンセントで使う。
② コンセント周りにほこりが付いていると発火。

![画像: コンセントとプラグ]

- ✗ 湿気や水たまりがある場所で使わない。

## チャイルドロックを設定する
（子供が閉じ込められるおそれ）

**電源「切」でドアが開かないようにします。**

1. メニュー
2. 開ける
3. チャイルドロック
4. 決定

![画像: QRコードのある様子]

- 「メニュー」を選択し、設定完了。
- 「解除」を押し「決定」を完了。

## 注意

### 次のことをしない
- ✗ 洗剤を混ぜない（カプセルのまま）
- ✗ やけど・感電・けがのおそれ
（洗濯ノートまたは）
- ✗ スポンジを使わない

### アースを取り付ける
（漏電時に感電のおそれ）
**専門業者に依頼してください。**
（安全規格に特技品は含まれていません）

### すぐに使用を中止する
（実際・停電・異常音・強い振動）
すぐに「電源」の操作を行い確認してください。

![画像: 電源のボタンの操作]

**異常時：動かない・異常音を発生したまま使う**
