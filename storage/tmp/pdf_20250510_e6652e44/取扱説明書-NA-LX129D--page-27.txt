# 自動投入 Q&A

## 自動投入で使う洗剤など

- **自動投入で使えないものは？**  
  - 次のものは使えません。手動投入でお使いください。 >> p28
    - 漂白剤、柔軟剤、用途外洗剤
    - 粉末洗剤、溶剤系洗剤  
    - 衣類用以外の洗剤、取扱説明に「全自動洗濯機で使用できないものの洗剤」の覚え書きされているもの  
- **洗剤などの翌週量とは？どこを見れば良い？ 【確認方法】
  - 洗濯量に応じ、水30 Lに対する使用量の目安です。  
  - 弊社製品問合せフォームをご確認ください。  
    洗濯物の量が使用量によって変化しやすけるうえ、洗濯メーカーにお問い合わせください。 >> 水30 Lに対する使用量
- **洗剤などの投入量・基準量を設定したいときどうする？**  
  - 自動投入できません。
  - 一度リセットしてから再設定してください。 >> p26
- **酸性や塩の種類を変えたい**
  - 次の手順に従ってください。
    - ① 自動投入タンクを抜き、手動入力する >> p50
    - ② 「酸性洗濯液または色付洗洗剤を含む理由」  
    - ③異なる場合、再設定する >> p24
- **おしゃか職洗剤を！？洗割タンクに入れてもいいの？**
  - 入っている洗剤タンクに書かれて有ります。 >> p26
    - ①洗剤タンク周辺が開放されていることを確認してください。
    - ②指定ブランドが上記で指定されているものの右より太２０スプーン（専用スプーンと印刷されているもの）をお使いください。
    - ③酸性洗剤ボトルは適用している。>> p27
  - ④香料がつけ有無、洗剤のところに投入されているます。  
- **漂白剤・汚れはがしを常に自動投入したい**
  - 対応できません。 >> p27
  - ③洗剤自動、ご利用の方での手洗い設定しております。
- **自動投入の後で洗液などを、間違って手動投入口に入れてしまったとき？**
  - 以下の手順をご確認ください。
    - ① 自動投入タンク・経路の手洗い設定する >> p50
    - ② ドラム内に破損しないで、手洗い設定してください>> p45
  - 洗剤が残らない場合（※必ずしてください。
    - 二重に設定され、入れる液体になります。

## タンクの残量

- **自動投入タンク内の残量を確認するには？**
  - 残量が少なくなると表示でお知らせします。 >> p25

[画像: 投入口のふた]

- **表示機,何回くらい洗濯できる？**
  - 洗剤、柔軟剤それぞれ>> p50
    - 洗剤：40回～180回
  - 柔軟剤：たき洗いする場合で変化するため、表示されない場合は、残量に応じて下さい。
- **タンクが空近い○印で表示されてない**
  - 無くなる前に補充してください。
- **自動投入タンクが清潔ネ成専反。何回位までできる？**
  - 定期的にタンクを開け、ぞうきけを掃除してください。  
    (確認後、ふたを閉めてください)  
- **一般衣類用洗剤250 mLで表示され作られる場合 **  
  - ① 50 ml：0回（斤持数目いっぱいで5 kg）の場合  
  - ② 100 ml：0回（斤持数目いっぱいで4 kg）の場合  
  - ③ 200 ml：20回（斤持数目いっぱいで5 kg）の場合  
- **印スットプボトル◟**
  - 1回は行いやすい>> 10回（ポリエテルスプレース）
    - 洗濯4kgを時間：12秒 >> 20回
  - ④ 透明ボトルに変更ている場合があります。
  - ※洗濯経路により異なります。必要なるお手数しました。
  - ※洗濯生綿、調整を手動入力。（洗剤800 g）>> 手動含まれる。
  - ※柔軟剤の手洗い準修二度手記する洗濯でご注意ください。（最低下記注意してください。）

##「自動投入タンク・経路」のお手入れ >> p50

- **3か月ごとに**
  - 1か月の間に自動投入を使わなかったとき
  - 洗剤のこびりつきを防ぐために手動洗い
    - 洗剤タンクの蓋を開け、手動で適度な量に起動してください。
  - 注意：高温多湿を手洗い設定する >> 紫外線や湿気について手洗い補充していただく。

## おかしいな？と思ったら

- **自動投入されないとき**
  - 自動投入洗剤設定されていないか？
  - 20秒設定されているか？
  - 制限されている買いなさいか？
  - ドラム内に柔軟剤投入したか？
  - 手洗いと設定されないか？
  - 自動投入方法： (予備洗剤なし：1分から: 45秒)そして十分なチェック後に再確認してください。
    - 経路指導後の下記取り扱いなさい。

### 廃除わかり♡
- 使っていくうちに自動投入タンクが変色した
  - 柔軟剤の設定引き継いでください。   
  - 定期的な清掃転在すい洗剤をつけることになっています。   
  - 使用上の指示と確認で下さい。（最適になる場合があります）
